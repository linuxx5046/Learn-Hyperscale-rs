# Guías de Aprendizaje - Hyperscale-RS

Conjunto completo de guías estructuradas para aprender Rust y comprender sistemas distribuidos a través del proyecto Hyperscale-RS.

## Contenido

Las guías están organizadas en diferentes formatos y niveles de profundidad:

- **`docs/`** - Documentación estructurada por tema
- Guías principales en formato markdown

## Índice de Guías

| # | Nombre | Descripción | Nivel | Tiempo |
|---|--------|-------------|-------|--------|
| 1 | [Guía de Aprendizaje](DETAILED_LEARNING_GUIDE_EN.md) | Guía progresiva desde fundamentos hasta implementación | Progresivo | 2-3h |
| 2 | [Análisis Técnico](HYPERSCALE_TECHNICAL_ANALYSIS_EN.md) | Análisis profundo de componentes técnicos | Avanzado | 1-2h |
| 3 | [Guía Completa](The%20Complete%20Learning%20Guide_%20Demystifying%20Hyperscale-RS.md) | Guía completa y exhaustiva | Intermedio | 2h |

## Lectura Recomendada

### Para Principiantes
1. Comienza con la **Guía de Aprendizaje**
2. Consulta los **Diagramas de Flujo** (en/flowcharts/)
3. Profundiza con el **Análisis Técnico**

### Para Aprendices de Nivel Intermedio
1. Lee la **Guía Completa**
2. Estudia los **Diagramas de Flujo** en detalle
3. Explora el código fuente del repositorio

### Para Aprendices Avanzados
1. Lee el **Análisis Técnico**
2. Estudia todos los **Diagramas de Flujo**
3. Contribuye con mejoras

## Relación con los Diagramas de Flujo

Las guías funcionan junto con los diagramas de flujo:

- **Guía de Aprendizaje** → Conceptos teóricos
- **Diagramas de Flujo** → Visualización de conceptos
- **Análisis Técnico** → Detalles de implementación

## Próximos Pasos

1. Elige tu nivel (Principiante/Intermedio/Avanzado)
2. Lee la guía correspondiente
3. Consulta los diagramas de flujo para visualizar
4. Explora el código fuente
